#### **1. Accessing the Reports Section**

- Navigate to the **Home Page**.
- Click on the **“reports”** tab.\
  This action will redirect you to the **Dashboard Interface**.

  ![image.png](/m1.png)

#### **2. Creating a New Report File or Folder**

Once on the dashboard:

- Click the appropriate button to **create a new file** or **create a new folder**, depending on your organizational needs.
- Enter a **meaningful name** for the file or folder in the naming field provided.
- Click **“Create”** to confirm and generate the item.

#### **3. Opening and Initializing the Report File**

After successfully creating the report file:

- Click on the newly created file to **open it**.
- At the **bottom-right corner** of the screen, locate the **Start icon button**.
- Click the **Start button** to initiate the configuration of the report file.

#### **4. Enabling Edit Mode**

To modify the contents of the file:

- Click on the **Edit icon button**, typically represented with a pencil or similar graphic.
- This activates edit mode, allowing structural and content-level modifications.

#### **5. Accessing the Widget Sidebar**

For enhanced customization and content elements:

- In the **upper-right corner** of the interface, you will see **three icon buttons**.
- Click the **second icon** to open the **Widget Sidebar**.

#### 6. Adding a Table Widget

From the widget sidebar:

- Locate and select the **“Tables Widget”**.
- Once selected, various options will appear.

#### 7. Configuring the Table Widget – Header Properties

The first section visible is **Widget Header Properties**.

**Header Properties Include:**

- **Always Display Header**: Toggle to always show the widget header.
- **Title Text**: Add a title to your table widget.
- **Font Size, Weight, Alignment**: Customize the appearance of your title.
- **Font Color**: Select the appropriate text color using the color picker.
- **Additional Properties**:
  - **Start and End Date** selectors
  - **Left and Right Logo Upload**: Upload logos or images on either side of the title by clicking “Choose File”.

    ![3 Pn](/3.png)

---

#### 8. Table Layout and Formatting

Upon clicking on the **table area**, a new panel titled **Table Properties** appears to the right.

**Editable Table Area:**

- Columns labeled A, B, C...
- Rows labeled numerically.

Clicking any cell activates cell editing and formatting options.

\*\*Table Properties – Text Formatting \*\*

Within the Table Text Formatter panel, you can configure:

- **Alignment**: Left, center, right.
- **Text Style**: Bold, italic.
- **Font Family and Size**: Choose your preferred typeface and text scale.
- **Border, Text, and Background Colors**: Customize color schemes for cells.

![5 Pn](/5.png)

#### 9. Configuring Table Data Properties

**Table Source and Columns:**

- **Select the data table**
- Choose relevant **columns** to display

**Grouping and Aggregation:**

- Grouping options allow data to be structured hierarchically.
- Aggregation options include **MIN, MAX, AVG, TOTAL** functions.

  ![6 Pn](/6.png)

**Sorting and Pagination:**

- Define sort criteria
- Customize number of rows displayed per page.

  ![7 Pn](/7.png)

---

#### 10. Column Formatting and Key Selection

**Key Selection**:

- Choose between **Auto** or **Manual** key assignment.
- Set the **column** used for unique identification (e.g., `shubhamtest.tag2`)
- Provide an optional **display name** for each column.

**Column Functions**:

- Enable summary values (e.g., **MIN** and **MAX**) per column.

**Custom Values**:

- Import or export Excel files to quickly configure static or dynamic column content.

  ![B9 Pn](/images/b9.png)

#### 11. Advanced Configuration – Styling and Conditional Formatting

**Data Formatting Options**

Under the **Data Formatting** section, users can define how numeric or time-based data should be presented.

There are three primary options in the dropdown:

1. **Decimal Point**
   - Allows users to specify the number of decimal places shown in numerical values.
2. **Timestamp**
   - Used when displaying time-based data.
   - Upon selection, a secondary dropdown **“Select Timestamp Format”** appears.
   - Choose from available standard formats such as `YYYY-MM-DD HH:mm:ss` or `MM/DD/YYYY`.
   - This ensures consistent and localized representation of time values.

     <img
       src="/images/reporttimestanmp.png"

     />
3. **Time Formatting**
   - A simplified format option where users can select how time values are displayed (e.g., hours:minutes)..

**Row or Cell Color Rules:**

- Add conditional formatting rules.
- Example: Apply **red color** to row if `tag2 > 225`.

  ![9 Pn](/9.png)

---

#### 12. Finalizing the Widget

- Once the widget is fully configured, click on the **“Add Widget”** button in the bottom-right corner.
- This embeds the customized table into your report layout.

#### Saving the Report

- After widget placement, click the **Save icon** located at the **bottom-right corner** of the screen to preserve your changes.
